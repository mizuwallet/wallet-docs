# MizuWallet

**MizuWallet** is a seamless next-generation wallet on the Aptos, that lets you build dapp more securely and easily. It implements the [Aptos Wallet Standard](https://github.com/aptos-labs/wallet-standard) and is designed to be used with the [Aptos Wallet Adapter](https://github.com/aptos-labs/aptos-wallet-adapter) or singly.

This developer documentation describes how to integrate Wallet into your dApp and enable your user to connect their MizuWallet to the Aptos blockchain.

## Quick Start

**MizuWallet** is now available on [aptos-wallet-adapter](https://github.com/aptos-labs/aptos-wallet-adapter). You can use it to connect your dApp to MizuWallet.
